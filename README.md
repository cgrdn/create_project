# create_project

This is a bash script which creates an empty project directory structure inspired by [Cookiecutter Data Science](https://drivendata.github.io/cookiecutter-data-science/).

The file structure is

```
   sample_project/
   ├── bin
   ├── data
   │   ├── external
   │   ├── interim
   │   ├── processed
   │   └── raw
   ├── docs
   ├── notebooks
   ├── README.md
   ├── references
   ├── reports
   │   └── figures
   └── src
```
